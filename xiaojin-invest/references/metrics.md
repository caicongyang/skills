# 估值指标详解

## P/E（市盈率）

- **公式**：Price / EPS
- **类型**：
  - Trailing P/E：基于过去12个月
  - Forward P/E：基于未来12个月预测
- **判断**：
  - < 15：合理
  - < 10：低估
  - < 5：严重低估（可能是机会）

## EV/EBITDA

- **公式**：Enterprise Value / EBITDA
- **优点**：排除财务杠杆和折旧摊销影响
- **判断**：比 P/E 更适合对比不同负债水平的公司

## EV/Sales

- **公式**：Enterprise Value / Revenue
- **适用**：高速增长但尚未盈利的公司
- **判断**：需结合行业平均

## EPS 增长分析

- **正常增长**：10-20%/年
- **高速增长**：30-50%/年
- **爆发增长**：>100%/年（往往是15-bagger特征）

## 机构预期

- **FactSet Consensus**：华尔街分析师加权平均
- **Bloomberg Consensus**：彭博一致预期
- ** Guidance**：公司自己给的业绩指引

## 15-bagger 特征

1. EPS 预测从低位大幅上调
2. Forward P/E 是个位数
3. AI/行业趋势明确
4. 机构一致性看好
5. 业务模式清晰可理解
