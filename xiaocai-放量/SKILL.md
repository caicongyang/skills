# xiaocai-放量

A股放量股分析工具。选出10个交易日内成交量翻倍且今日放量的股票。

## 配置

数据库配置通过 `config/database.json` 管理：

```json
{
  "host": "43.133.13.36",
  "port": 3333,
  "user": "root",
  "password": "root",
  "database": "stock"
}
```

## 执行

```bash
python3 analyze.py
```

## 筛选条件

- **今日成交量 ≥ 2倍昨日成交量**
- **今日成交量 ≥ 2倍10日前成交量**

两个条件同时满足才算符合。

## 输出

按概念板块分组，展示股票代码、名称、收盘价、涨幅、今日/昨日比率、今日/10日前比率。
